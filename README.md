# 终末地游戏公告推送插件

## 插件简介

本插件用于自动监听并推送《终末地》(Endfield) 游戏的国服官方公告。支持监听 Windows、iOS、Android 三个平台的公告，当有新公告发布时自动推送到已开启推送的群组。

## 功能特性

- 支持三平台监听（Windows、iOS、Android）
- 可自由选择监听的平台（支持多选）
- 自动检测新公告并推送
- 时间过滤：只推送最近 N 天内的公告
- 防重复推送：已推送的公告不会重复发送

## 使用指令

### 管理员指令

- `兔兔开启终末地公告推送` - 在当前群组开启公告推送功能
- `兔兔关闭终末地公告推送` - 在当前群组关闭公告推送功能
- `兔兔测试终末地公告` - 手动检查并获取最新公告（测试用）

### 普通指令

- `兔兔查看推送平台` - 查看当前启用的推送平台列表
